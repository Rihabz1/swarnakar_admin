# Swarnakar Admin

Flutter Web admin panel for the Swarnakar jewelry app. The UI is responsive for
phone and desktop, and integrates Firebase Auth + Firestore with Riverpod and
go_router.

## Prerequisites

- Flutter SDK (3.7+)
- Firebase CLI and FlutterFire CLI (`firebase` and `flutterfire`)

## Setup

1) Install packages:

```bash
flutter pub get
```

2) Configure Firebase (generates `lib/firebase_options.dart`):

```bash
flutterfire configure --project=swarnakar-59dd6
```

3) Ensure Firestore documents follow the required schema and admin users have
`users/{uid}.role == "admin"`.

## Run (Web)

```bash
flutter run -d chrome
```

## Build (Web)

```bash
flutter build web
```

## Notes

- Non-admin users are signed out immediately after login with an access denied
	screen.
- `firebase_options.dart` is intentionally not committed; it is generated by
	FlutterFire.
